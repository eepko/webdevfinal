Introduction

My portfolio site represents a professional version of myself, showcasing my skills, projects, and achievements. It addresses potential employers and clients looking for a skilled and creative individual. I chose a clean and intuitive structure to effectively showcase my work and ensure easy navigation.

Inspiration

1. Apple - I was inspired by their sleek and minimalist design, which emphasizes high-quality visuals and clear, concise content.

2. Awwwards - This site influenced me by showcasing a diverse range of innovative and artistic web designs, which helped me understand current design trends.

3. Behance - I found their portfolio presentations particularly useful in designing my own site, as they emphasize high-quality visual presentations and creative layouts.

Accessibility

1. The use of high-contrast colors helps users with visual impairments distinguish between different sections and content on the site.

2. Alternative text is provided for all images to assist screen readers, ensuring that visually impaired users can understand the content of the images.

3. Keyboard navigation is fully supported, allowing users with mobility impairments to navigate the site without relying on a mouse.

Usability

1. The navigation menu is simple and intuitive, allowing users to easily find information and access different sections of the site.

2. Responsive design ensures the site works well on both desktop and mobile devices, providing a seamless experience across different screen sizes.

3. Clear calls-to-action guide users through the site effectively, helping them understand what actions to take next.

Learning

1. I learned how to use CSS Grid for layout purposes by following online tutorials and experimenting with different grid configurations.

2. I researched accessibility best practices to ensure my site is usable for everyone, including those with visual, auditory, and mobility impairments.

3. I explored different JavaScript libraries to enhance site functionality and user interaction, such as adding animations and dynamic content.

Evaluation I

I believe the most successful aspects of my work are the clean and modern design, which effectively showcases my portfolio while maintaining a professional appearance. This is because the design aligns well with current web design trends and emphasizes user experience.

Evaluation II

Aspects that could be improved include adding more interactive elements to engage users and make the site more dynamic. In the future, I might incorporate more advanced features such as interactive infographics or video content to further enhance user engagement.

Resources

1. W3Schools - I used W3Schools for learning and referencing various HTML and CSS techniques.

2. YouTube - I followed several coding tutorials on YouTube to learn JavaScript and advanced CSS techniques.

3. MDN Web Docs - I referred to MDN Web Docs for in-depth explanations of web technologies and best practices.

4. Code or tools used: I utilized various libraries and tools such as CSS Grid and Flexbox for layout, and included comments in my code to indicate their usage.
